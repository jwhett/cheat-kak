# CHEAT-KAK

Highlighter for cheatsheets used with [`navi`](https://github.com/denisidoro/navi).


### Installation

Install with [`plug`](https://github.com/andreyorst/plug.kak):

```
# In ~/.config/kak/kakrc
plug "jwhett/cheat-kak"
```

and `:plug-install`
